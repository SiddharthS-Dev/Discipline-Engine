# 📖 Overview

Execution Discipline Engine is an enterprise-grade employee management platform built to improve organizational discipline, transparency, and productivity.

The platform enables organizations to manage employee activities through structured workflows, performance tracking, role-based approvals, audit logging, and automated score management.
